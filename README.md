# bfcp
Simple Brainf**k Compiler written in Rust.

## Installation
From source:
- install a recent Rust toolchain
- clone this repository and run `cargo install --path .`

## Usage

```
Usage: bfcp [options] [file]

Options:
    -h, --help          Print help and exit
    -o, --output filename
                        Output file name
    -b, --buffersize 30000
                        Length of the brainf**k buffer
```
